Prerequisites:

  • This tutorial assumes very basic knowledge of how to use the terminal & shell (commands like cd and ls).
  • This tutorial is written on Mac OSX. If you are on a Windows or linux machine note that some material may be different (although git/GitHub terminology and commands should remain consistent across operating systems).

What is Git?

Git is a version control system. This means that git is an extra tool that you can use as a developer to track changes and manage shared code between a group of people. Here are some examples of what Git enables you to do:

  • You can have multiple different versions of your code
    • For example, you versions have two copies of code, one for the last fully-working update of your app and a second where you actively develop and debug new features.
  • Git also allows you to roll-back to a previous version of your code if necessary.
    • Many young developers will comment out code instead of deleting it, for the fear that they may need to bring back the code at a later time. Git allows you to just delete any unneeded code and restore it later, if needed.
    • You don't have to cmd-z countless times to get back to a working version of your code; just use git to reset your code back to the latest working version.
  • Git makes it much easier to share code between multiple people in a team (we'll get into how at a later time).

Installing Git:

Installing git is as easy as openning up a terminal window and running the command "git --version". If git is not already installed on your machine, a pop-up will appear asking if you would like to install git, press Install when prompted.

  • git --version

What is Github?

Github is an online service that allows you to create a 'remote repository' for your team's code, think of this like a shared Google Drive folder, but specifically designed for code.

Signing up for GitHub:

Signing up for GitHub is as easy as navigating to GitHub's website and following the directions on screen.
Note: GitHub offers some nice (and free) additional features for students. If you're a student, I strongly recommend getting the Student Developer Pack after creating your account.
