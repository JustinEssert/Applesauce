Intro to Git and GitHub

What is Git?

Git is a version control system. This means that git is an extra tool that you can use as a developer to track changes and manage shared code between a group of people. Here are some examples of what Git enables you to do:

  • You can have multiple different versions of your code
    • For example, you could have a copy of your code that you know works, as well as a version where you are adding a new feature, but the code does not work fully yet.
    • This means that you don't need to comment out code that you don't want to delete, just delete it and if you need it later, you can still access it.
  • Git also allows you to roll-back to a previous version of your code if necessary.
  • Git makes it much easier to share code between multiple people in a team.

What is Github?

Github is an online service that allows you to create a 'remote repository' for your team's code, think of this like a shared Google Drive folder, but specifically designed for coding.
